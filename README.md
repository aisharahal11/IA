# IA
que es la ia
que es el machine learning, tipos de ml(diferentes aprendizajes)

1) La Inteligencia Artificial (IA) es una rama de la informática que se dedica a crear sistemas o programas capaces de realizar tareas que normalmente requieren inteligencia humana

2) El Machine Learning (en español aprendizaje automático) es una parte de la Inteligencia Artificial que permite que las computadoras aprendan a partir de datos sin que alguien tenga que programar todas las reglas paso a paso.

3) Los principales tipos de aprendizaje en Machine Learning son:

Aprendizaje supervisado
El modelo se entrena con datos que ya tienen la respuesta correcta (etiquetas). Aprende la relación entre entrada y salida para poder predecir resultados nuevos.

Aprendizaje no supervisado
El modelo recibe datos sin etiquetas y busca patrones o estructuras por sí mismo, como agrupar elementos similares.

Aprendizaje por refuerzo
Un agente aprende interactuando con un entorno. Recibe recompensas o castigos según sus acciones y mejora su estrategia con el tiempo.

Aprendizaje semi-supervisado
Combina datos etiquetados y no etiquetados para entrenar el modelo.

Aprendizaje auto-supervisado
El modelo genera sus propias tareas de aprendizaje a partir de los datos, creando automáticamente las etiquetas necesarias.
